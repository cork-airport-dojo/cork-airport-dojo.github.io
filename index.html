Cork Airport Dojo is hosted and supported by IBM Ireland.

Cork Airport Dojo is a coding club dedicated to inspiring and enabling young minds to explore the world of coding and technology. We are committed to fostering a vibrant and inclusive learning environment where young enthusiasts can develop their coding skills and unleash their creative potential.

Current Classes

We host a wide variety of different classes for both newcomers and intermediate programmers alike. Classes will take place on Saturdays between 14:00 to 15:20 and 15:40 to 17:00.

You can come for just the class you want. We take a break at around 15:30 between classes, which can be a good opportunity to arrive for the second class, or leave after the first class.

Time Class name Description Experience
14:00 - 15:20 Beginner's Python Introduction to programming fundamentals Beginner
Beginner's HTML Introduction to website development Beginner
Introduction to C++ Introduction to a difficult, statically-typed language. Intermediate
15:40 - 17:00 Intermediate Python Continuing with development of Python skills (Object-Oriented Programming) Intermediate
Intermediate JavaScript Continuing to learn the scripting language of the web Intermediate
Advanced Projects A chance for more experienced coders to develop projects that can help them grow, and put these projects in their CV Experienced
