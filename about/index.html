Getting to class

CoderDojo @ IBM Cork takes place on IBM premesis in the Red Hat building in the Cork Airport Business Park. You will see lots of Red Hat branding, but don't worry, you're in the right place. The bottom floor is for IBM offices, where we will host classes. The office is just down the road from the Cork International Hotel.

Directions: 6700 Avenue 6000, Lehenagh More, Cork, T12 XR60

If your kid is younger than 13, you (the parents / guardians) will need to stay in the building during their class(es). There is a kitchen where you can relax and have a tea or coffee and chat withother parents while you wait. Alternatively, you can sit in with the kids and take part in the class with them.

Classes

We host a wide variety of different classes for both newcomers and intermediate programmers alike. Classes will take place between 14:00 to 15:30 and 15:30 to 17:00.

Normally, beginner's classes will take place in the earlier time slot. Normally, we run some of the following classes, detailed below

Experience level Class name Notes
Beginner Beginner's HTML Introduction to web development
Beginner Advanced HTML Generally for those who've completed Beginner's HTML, expands on previous experience.
Intermediate Introduction to programming Introduction to either Python 3 or JavaScript, with the aim to solve problems, and complete projects.
Intermediate Data structures & algorithms Using Python 3 to design algorithms and introduction to optimizing algorithms for computation and memory complexity. Will also include advanced project work, excellent for projects for a developer's resume.
Advanced Game development Using previous experience with User Interface and advanced algorithms and data structures, start learning how games work and design a simple game.

We are very open to suggestions on new classes to try out, and have mentors with a broad range of experience, and are very capable to teach.

What you need to bring

  • a laptop
  • a lunch, if you like
  • a parent - very important read more

We recommend that if you can, install Visual Studio Code (important, don't install Visual Studio - similar name, very different programs). Visual Studio Code (or VSCode) is a text-editor, something programmers use to write code, and using a good one can help children learn coding fundamentals faster.

Belts and grading

At CoderDojo, when kids reach a milestone in their learning, they receive a certificate and a new belt (a USB-stick with increasing storage the higher the belt). Generally, completing a semester of a class will yield a new belt for a student.

We previously used end-of-semester tests to gauge kids on what they've learned, but going forward, we will base each kids' progression off a project they complete during the semester for that class. For example; a kid who completes a website project for the Beginners' HTML class will receive a white belt (the first one).

Safeguarding

CoderDojo's safeguarding policy aims to protect protect children, young people, and vulnerable adults who take part in CoderDojo activities.

All CoderDojo @ IBM Cork mentors will have completed CoderDojo's safeguarding module, which covers the following topics;

  • code of behavior
  • child disclosures
  • online safety
